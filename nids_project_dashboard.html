Training
Random Forest
Train a RandomForestClassifier with 100 trees and class_weight="balanced" to handle the normal/attack imbalance. Uses all CPU cores (n_jobs=-1). Typical training time: 5–20 seconds on 50k samples.
from sklearn.ensemble import RandomForestClassifier clf = RandomForestClassifier( n_estimators=100, max_depth=20, n_jobs=-1, class_weight="balanced" ) clf.fit(X_train, y_train)

Evaluation
Confusion matrix
Evaluate with accuracy, ROC-AUC, and a full confusion matrix. The matrix shows true/false positives and negatives — critical for understanding missed attacks (FN) vs false alarms (FP).
from sklearn.metrics import confusion_matrix, roc_auc_score cm = confusion_matrix(y_test, y_pred) auc = roc_auc_score(y_test, clf.predict_proba(X_test)[:,1]) tn, fp, fn, tp = cm.ravel() # tp = caught attacks, fn = missed attacks!

Live detection
Scapy
nids_live.py opens a raw socket with Scapy, extracts 41 KDD-style features from each packet, and classifies in real-time. Run with --sim flag for simulation without root access.
from scapy.all import sniff sniff( iface="eth0", filter="ip", prn=lambda pkt: classify_packet(pkt, clf, scaler), store=False ) # Each packet → feature vector → RF model → Normal/Attack
